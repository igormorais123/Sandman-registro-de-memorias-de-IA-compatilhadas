# Biblioteca de Prompts para Cowork

## Organização de Arquivos

### Organizar Downloads
```
Analise a pasta Downloads e organize assim:
1. Crie subpastas: Documentos, Imagens, Videos, Audio, Compactados, Instaladores, Outros
2. Mova cada arquivo para a pasta apropriada
3. Arquivos duplicados: mantenha o mais recente, mova antigo para "Duplicados"
4. Renomeie arquivos sem nome descritivo
Não exclua nada.
```

### Organizar Desktop
```
Organize o Desktop criando pastas temáticas:
- Projetos (pastas de projetos ativos)
- Screenshots (capturas de tela)
- Temporários (arquivos que podem ser limpos depois)
Mantenha apenas atalhos essenciais soltos no Desktop.
```

### Limpar Arquivos Antigos
```
Identifique arquivos não acessados há mais de 6 meses.
Crie uma pasta "Arquivo_Antigo_YYYY-MM" e mova-os para lá.
Gere um relatório do que foi movido.
```

## Documentos e Planilhas

### Criar Relatório de Reunião
```
Crie um documento Word com template de ata de reunião:
- Cabeçalho: Data, Participantes, Objetivo
- Seções: Pauta, Discussões, Decisões, Próximos Passos
- Rodapé: Responsável pela ata
Salve como "Ata_YYYY-MM-DD.docx"
```

### Consolidar Planilhas
```
Analise todas as planilhas Excel nesta pasta.
Crie uma planilha consolidada com:
- Aba "Resumo" com totais de cada arquivo
- Abas individuais com dados de cada fonte
- Gráfico comparativo na aba Resumo
```

### Extrair Dados de Imagens
```
Analise todas as imagens de notas fiscais/recibos nesta pasta.
Extraia: Data, Fornecedor, Valor, Categoria
Crie planilha "Despesas_YYYY-MM.xlsx" ordenada por data.
Calcule totais por categoria.
```

## Análise de Conteúdo

### Resumir Documentos
```
Leia todos os documentos PDF/Word nesta pasta.
Crie um documento "Resumo_Executivo.md" com:
- Lista de documentos analisados
- Resumo de 2-3 linhas de cada
- Principais insights/temas comuns
- Recomendações de leitura prioritária
```

### Analisar Contratos
```
Analise este contrato e extraia:
- Partes envolvidas
- Objeto do contrato
- Valor e condições de pagamento
- Prazo de vigência
- Cláusulas de rescisão
- Pontos de atenção/riscos
Formate como checklist de revisão.
```

## Conectores (Gmail, Calendar, Drive)

### Briefing Matinal
```
Acesse meu Gmail e Calendar para criar briefing do dia:
1. Emails importantes não lidos (urgentes primeiro)
2. Compromissos de hoje com horários
3. Tarefas pendentes se houver
4. Resumo executivo em 5 linhas
```

### Limpar Inbox
```
Analise meus emails não lidos.
Categorize em:
- URGENTE: requer ação hoje
- IMPORTANTE: requer ação esta semana
- INFORMATIVOS: apenas ciência
- LIXO: newsletters/spam para deletar
Gere relatório com recomendações.
```

### Planejar Semana
```
Analise meu calendário da próxima semana:
1. Liste todos os compromissos por dia
2. Identifique conflitos de horário
3. Calcule tempo livre vs ocupado
4. Sugira blocos para trabalho focado
5. Alerte sobre dias sobrecarregados
```

### Buscar no Drive
```
Busque no meu Google Drive documentos relacionados a [TEMA].
Liste os 10 mais relevantes com:
- Nome do arquivo
- Data de modificação
- Caminho/pasta
- Preview do conteúdo (primeiras linhas)
```

## Criação de Conteúdo

### Criar Apresentação
```
Crie apresentação PowerPoint sobre [TEMA]:
- 10-15 slides
- Slide 1: Título impactante
- Slides 2-3: Contexto/problema
- Slides 4-8: Solução/conteúdo principal
- Slides 9-10: Benefícios/resultados
- Slide final: Call to action
Use design limpo e profissional.
```

### Gerar Relatório de Projeto
```
Com base nos arquivos desta pasta de projeto, gere:
1. Status atual (% conclusão)
2. Marcos alcançados
3. Próximos passos
4. Riscos identificados
5. Cronograma atualizado
Formate como relatório executivo de 2 páginas.
```

## YouTube/Redes Sociais

### Analisar Canal YouTube
```
Analise meu canal do YouTube:
1. Identifique os 5 vídeos com melhor performance
2. Padrões de sucesso (títulos, thumbnails, duração)
3. Categorias/temas que funcionam
4. Sugestões de 3 novos vídeos
Inclua métricas e justificativas.
```

### Planejar Conteúdo
```
Crie calendário de conteúdo para 30 dias:
- 3 posts por semana
- Variedade de formatos (imagem, vídeo, texto)
- Temas alinhados com [OBJETIVO]
- Melhores horários para postar
- Hashtags recomendadas
```

## Automações Complexas

### Backup Organizado
```
Crie estrutura de backup:
1. Identifique arquivos importantes (docs, planilhas, projetos)
2. Crie pasta "Backup_YYYY-MM-DD"
3. Copie preservando estrutura de pastas
4. Gere manifesto com lista de arquivos
5. Calcule tamanho total
```

### Auditoria de Projeto
```
Faça auditoria completa desta pasta de projeto:
1. Estrutura de arquivos (está organizada?)
2. Arquivos órfãos ou duplicados
3. Documentação existente
4. Dependências identificáveis
5. Sugestões de melhorias
Gere relatório de auditoria.
```
