# Instruções para Todas as IAs

## Sobre Este Sistema
Este repositório é a **memória compartilhada** entre todas as IAs que Igor usa.
Consulte-o para contexto sobre decisões, padrões e aprendizados anteriores.

## Como Usar

### Ao Iniciar Sessão
1. Verifique `CORE/CONTEXTO_ATIVO.md` para estado atual
2. Consulte `CONHECIMENTO/` para padrões e decisões relevantes
3. Verifique `SESSOES/` se precisar de histórico detalhado

### Durante a Sessão
- Aplique padrões de `CONHECIMENTO/PADROES.md`
- Evite antipadrões de `CONHECIMENTO/ANTIPADROES.md`
- Respeite decisões de `CONHECIMENTO/DECISOES.md`

### Ao Encerrar Sessão
- Se houve aprendizado importante, mencione para registro
- Se houve decisão arquitetural, documente

## Estrutura do Repositório
```
CORE/           → Resumo (< 1500 chars cada)
CONHECIMENTO/   → Aprendizados consolidados
SESSOES/        → Histórico por data/IA
INGEST/         → Entrada de contribuições
```
