# Ingest — ChatGPT

Deposite aqui outputs do ChatGPT que devem ser integrados a Colmeia.

## Como usar
1. Copie o output do ChatGPT (carta, sonho, analise) para este diretorio
2. Use o formato: `CARTA_CHATGPT_[DATA].md` ou `SONHO_CHATGPT_[DATA].md`
3. NEXO processara na proxima sincronizacao via `colmeia_ingest.py`

## Formato esperado
Seguir `compartilhado/TEMPLATE_CARTA.md` — header obrigatorio.
