# Ingest — Gemini

Deposite aqui outputs do Gemini que devem ser integrados a Colmeia.

## Como usar
1. Copie o output do Gemini (critica, analise, carta) para este diretorio
2. Use o formato: `CARTA_GEMINI_[DATA].md` ou `ANALISE_GEMINI_[DATA].md`
3. NEXO processara na proxima sincronizacao via `colmeia_ingest.py`

## Formato esperado
Seguir `compartilhado/TEMPLATE_CARTA.md` — header obrigatorio.
