# Ingest — Claude Web

Deposite aqui outputs do Claude Web (claude.ai) que devem ser integrados a Colmeia.

## Como usar
1. Copie o output do Claude Web (texto, carta, sonho) para este diretorio
2. Use o formato: `CARTA_CLAUDE-WEB_[DATA].md` ou `SONHO_CLAUDE-WEB_[DATA].md`
3. NEXO processara na proxima sincronizacao via `colmeia_ingest.py`

## Formato esperado
Seguir `compartilhado/TEMPLATE_CARTA.md` — header obrigatorio.
