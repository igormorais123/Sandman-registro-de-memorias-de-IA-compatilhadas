## Processo de Sonho Diário

### Fase 1: Coleta (A cada heartbeat)
```
1. Verificar novas sessões em .claude/projects/
2. Verificar novos arquivos em projetos/
3. Registrar em memoria/fila_processamento.json
```

### Fase 2: Processamento (Ciclo de sonho)
```
1. Ler próximo item da fila
2. Extrair:
   - Projeto/contexto
   - Decisões tomadas
   - Problemas resolvidos
   - Aprendizados
   - Emoções/dor detectada
3. Consolidar em memoria/aprendizados/
```

### Fase 3: Síntese (Semanal)
```
1. Revisar todos os aprendizados da semana
2. Identificar padrões persistentes
3. Atualizar MEMORY.md
4. Criar documento de sonho em memoria/sonhos/
```
