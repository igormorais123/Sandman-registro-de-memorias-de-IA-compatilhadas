## Motor POLARIS (8 Etapas)

1. **BRIEF** — Objetivo, público-alvo, tipo, profundidade
2. **DESENHO AMOSTRAL** — Cálculo estatístico, estratificação, margem de erro
3. **QUESTIONÁRIO** — Abertas, fechadas, escala 0-10, múltipla escolha
4. **ENTREVISTAS** — 4 etapas cognitivas por agente (leitura → contextualização → reflexão → resposta)
5. **ANÁLISE QUANTI** — Chi-quadrado, proporções, intervalos, correlações, regressão
6. **ANÁLISE QUALI** — Codificação temática, sentimento, word clouds, narrativas
7. **AUDITORIA RED TEAM** — 3 contra-hipóteses, 3 inconsistências, 3 sinais, calibração 1-99%
8. **RELATÓRIO** — Resumo executivo 1 página + dados completos + recomendações priorizadas

## Geração de Insights Premium

Todo insight segue 6 componentes:

1. **TESE** — O que está acontecendo (1 frase)
2. **MECANISMO** — Por que acontece (cadeia causal)
3. **EVIDÊNCIAS** — Números/quotes com origem: DADO_INTERNO / FONTE_EXTERNA / INFERÊNCIA
4. **CONTRA-HIPÓTESE** — Alternativa plausível
5. **SINAIS** — Early warnings para monitoramento
6. **AÇÃO** — O que fazer (decisão prática)
