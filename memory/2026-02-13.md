## Correção Importante: Serviço Gateway

**Problema:** Igor reportou quedas frequentes do gateway ("vc ainda esta caindo mt")

**Diagnóstico:**
- `openclaw status` mostrava: "Gateway service: systemd installed · disabled · stopped"
- O gateway estava rodando como processo solto, não como serviço
- Qualquer instabilidade fazia cair sem reinício automático

**Solução aplicada:**
```bash
openclaw gateway install
systemctl --user enable openclaw-gateway
systemctl --user start openclaw-gateway
```

**Resultado:** Serviço agora está `active (running)` e `enabled`
- Reinício automático se cair
- Sobrevive a reinícios do WSL (com lingering habilitado)
